# JavaScript Basic Questions

### 1️⃣ What is the difference between var, let, and const?

- **var**: Old way to declare variable. It can be re-declared and updated.
- **let**: Modern variable. It can be updated but cannot be re-declared in the same scope.
- **const**: Constant variable. It cannot be updated or re-declared.

---

### 2️⃣ What is the spread operator (...)?

The **spread operator (...)** is used to copy or expand arrays or objects.

Example:
const numbers = [1,2,3];
const newNumbers = [...numbers,4,5];

---

### 3️⃣ What is the difference between map(), filter(), and forEach()?

- **map()**: Creates a new array after modifying each element.
- **filter()**: Creates a new array with elements that pass a condition.
- **forEach()**: Loops through an array but does not return a new array.

---

### 4️⃣ What is an arrow function?

Arrow function is a shorter way to write a function in JavaScript.

Example:
const add = (a,b) => a + b;

---

### 5️⃣ What are template literals?

Template literals are used to write strings with variables using backticks (` `).

Example:
const name = "Fahim";
console.log(`Hello ${name}`); -->
